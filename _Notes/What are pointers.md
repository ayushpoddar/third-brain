- A pointer has two aspects - value and type.
	- **Value**: It is the address at which some object is stored
	- **Type**: Indicates the kind/type of object stored at that location

## Declaring pointers
```c
T *p; // Declares a pointer p of data type T

// Example
char *p;
```
